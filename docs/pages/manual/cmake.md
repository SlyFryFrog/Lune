# CMake

This section covers CMake-related topics for the core engine. For details on specific extensions, refer to their respective documentation pages.

## Flags and Options

- `"BUILD_DEMO"`: Enables building the demo project set to `ON`, defaults to `OFF`.
- `"LYS_CUSTOM_MAIN"`: Builds Lys as a shared library when `ON`, defaults to `OFF`.
